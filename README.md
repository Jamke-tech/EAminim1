# EAminim1
Hi han dues carpetes una de backend i una de frontend on estan separats les coses

## Inicialització del documents
### Backend
S'activarà en el localhost:3600

```bash
cd backend
 npm run dev
```

### Frontend
S'activarà en el localhost:4200 

```bash
cd frontend/Angular/Minim1EA

ng serve

```
